honeybadger

-----

  >>> from honeybadger import HoneyBadger
  >>> hb = HoneyBadger()
  >>> hb.runInSlowMotion()
  1
  while hb.is_hungry:

  ... hb.eatSnake('cobra')
  ... assert hb.is_gross
